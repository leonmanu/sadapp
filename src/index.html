Buscador Inteligente de Documentos

Acceso a Contenido Indexado en Google Drive

Naturaleza de la Búsqueda

  • 1.

    **Búsqueda en Contenido Diverso:** La herramienta utiliza la API de Drive para escanear tanto el nombre como el **contenido interno** de diversos formatos de archivo (PDFs, DOCX, hojas de cálculo, etc.).

  • 2.

    **Desafío del OCR (Documentos Ilegibles):** La búsqueda por contenido puede fallar en documentos escaneados o muy antiguos, ya que el motor de Google no pudo generar una capa de texto indexable (OCR). La aplicación está diseñada para **identificar** estos documentos como "ilegibles".

  • 3.

    **Cómo Optimizar su Búsqueda:**

    • Use la barra de búsqueda para términos clave por **nombre o contenido**.
    • Preste atención a los archivos marcados como **"Ilegibles"** en los resultados.
    • Para estos archivos, use la opción **"Re-indexar"** que fuerza a Google a intentar el proceso de OCR nuevamente para que el contenido sea buscable.
